# WC2026 Predictor

Predict the FIFA World Cup 2026 with data, not guesswork.

WC2026 Predictor is a web application that generates match predictions using machine learning models trained on historical football data. Every fixture includes estimated probabilities for a home win, draw, or away win, along with confidence scores and model explanations.

---

## The Idea

Predictions are produced by trained models that analyze past matches and engineered features to estimate the likelihood of each possible outcome. As real matches conclude, those predictions can be compared against reality to evaluate model performance.

---

## What You'll See

* Upcoming World Cup fixtures
* Machine learning predictions for every match
* Home win, draw, and away win probabilities
* Leaderboards comparing model performance
* Tournament standings and knockout progression

---

## Not AI-Generated Guesses

The predictions are not produced by a large language model or chatbot.

They are generated by dedicated machine learning models trained on football data and evaluated using quantitative metrics. The website simply visualizes those predictions and makes them easy to explore.

---

## Why This Project Exists

WC2026 Predictor was built to make sports forecasting transparent and interactive, allowing users to inspect probabilities, compare competing models, and follow how predictive performance evolves throughout the tournament.
